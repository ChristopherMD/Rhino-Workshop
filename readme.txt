# Rhino Workshop Website Read Me File.

**************************************

## Description

Rhinoworkshop.com is the personal website of Christopher Doud.
Coded in 2020 using Visual Studio Code.
Languages:
HTML5 
CSS3 - flexbox
Bootstrap
Javascript ES6

**************************************

## Features
 - Reactive design - adjusts to all screen sizes and browsers.
 - Light and Dark Mode - Customized light and dark page format.
 - Highlight Color Changer - Customized color picker for page highlight color.
 - Gallery - Customized gallery on parameters.
 - Quiz Game - 7 random questions from a pool of 20 tracking score.
 - Joke Teller - API collecting using async functions
 - Snake Game - javaScript snake apple game.
 - Texas Lottery Quick Picker+ - random number generator using numerology for 4 Tx Lotto games.
 - Contact Form in Javascript
